# Project status

This project is designed to be reproducible on the author's local machine. It regenerates the final manuscript, tables, and figures.

Submission-readiness checklist still requiring human confirmation:

- final author order
- affiliations
- corresponding author email
- funding statement
- conflict-of-interest statement
- target journal reference style and word limits
- final approval from coauthors
- confirmation that all third-party data terms permit the intended sharing route
